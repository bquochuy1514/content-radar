# Content Radar

A pipeline that scrapes a support knowledge base, converts articles to
clean Markdown, and syncs them into an AI-powered support assistant
(Gemini File Search) — kept up to date via a daily automated job.

## Setup

1. Clone this repo
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `.\venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Mac/Linux)
4. Install dependencies: `pip install -r requirements.txt`
5. Copy `.env.sample` to `.env` and fill in your `GEMINI_API_KEY`

## How to run locally

python main.py
